# Shake Tweaks

A small QOL mod that lets you alter the game's shake mechanics for the screen and hand.

### Features

- Disable Screen Bobbing
- Disable Hand Bobbing
- Disable Map Bobbing
- Disable Screen Damage Tilt
- Disable Hand Damage Tilt
